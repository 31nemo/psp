# 📀 Web PSX PBP Converter (Multi-Disc & High Compatibility)

브라우저에서 직접 실행되는 강력한 PS1 게임용 PBP 변환 도구입니다. 서버 없이 100% 로컬에서 작동하며, 멀티 디스크 병합 및 에뮬레이터 호환성에 최적화되어 있습니다.

## 🚀 주요 기능

### 1. 멀티 디스크 지원 (Multi-Disc Merge)
- 최대 5장의 CD를 하나의 PBP 파일로 통합할 수 있습니다.
- PSP 실기 및 에뮬레이터에서 원활한 '디스크 교체' 기능을 지원하는 표준 구조를 따릅니다.

### 2. 폭넓은 포맷 지원 및 자동화
- **지원 포맷**: CUE+BIN, ISO, IMG 파일을 지원합니다.
- **BIN/ISO/IMG 단독 변환**: CUE 파일이 없어도 게임 데이터 파일만 올리면 자동으로 표준 CUE 구조를 생성하여 변환합니다.
- **자동 제목 추출**: 파일명에서 게임 제목과 Game ID를 자동으로 분석하여 입력 필드를 채워줍니다.

### 3. 에뮬레이터 호환성 극대화 (RetroArch, PPSSPP 등)
- **표준 레이아웃 보정**: 상용 PBP와 동일한 바이너리 오프셋을 준수하여 에뮬레이터 실행 실패 문제를 해결했습니다.
- **기본 실기 엔진 내장**: Base PBP가 없더라도 PSP 엔진(`DATA.PSP`)을 자동으로 삽입하여 즉시 실행 가능한 파일을 만듭니다.
- **SFO 표준화**: 호환성이 가장 검증된 `v1.00` 규격을 기본값으로 사용합니다.

### 4. 커스텀 리소스 및 PSN 파일 지원
- **리소스 교체**: 게임의 아이콘(`ICON0`), 배경(`PIC1`), 배경음악(`SND0`) 등을 자유롭게 변경하거나 Base PBP에서 추출할 수 있습니다.
- **PSN 부가 파일 통합**: `KEYS.BIN` 및 `DOCUMENT.DAT` 파일을 함께 등록하여 관리할 수 있습니다.
- **Base PBP Repack**: 게임 이미지를 새로 넣지 않고 기존 PBP의 리소스만 교체하는 기능도 지원합니다.

### 5. 100% 프라이빗 & 안전
- 모든 처리는 사용자의 브라우저 메모리 내에서 이루어집니다.
- 파일이 서버로 전송되지 않으므로 개인정보 및 데이터 보안이 완벽하게 유지됩니다.

## 🛠 사용 방법

1. **디스크 슬롯**: 게임 파일(CUE+BIN, BIN, IMG, ISO 중 하나)을 등록합니다. 멀티 디스크라면 '+ 디스크 추가'로 슬롯을 늘릴 수 있습니다.
2. **Base PBP (권장)**: 더 완벽한 호환성을 원하신다면 작동이 검증된 Base PBP를 등록하세요. (미등록 시 내장 엔진 사용)
3. **커스텀 리소스**: 아이콘이나 배경 이미지를 변경하려면 해당 슬롯에 파일을 드래그합니다.
4. **변환**: '⚡ PBP로 변환' 버튼을 누르면 즉시 변환이 시작되며 완료 후 다운로드 링크가 생성됩니다.

## 💡 Tip
- RetroArch 등에서 실행이 안 된다면, 잘 작동하는 다른 PBP 파일을 **②번 Base PBP** 칸에 넣어보세요. 해당 파일의 엔진을 복제하여 호환성을 100%로 끌어올립니다.
